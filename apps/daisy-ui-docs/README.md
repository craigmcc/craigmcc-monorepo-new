# @repo/daisy-ui-docs

Public-facing docs app for `@repo/daisy-ui` focused on component API contracts and machine-readable metadata.

## Scripts

- `pnpm dev`
- `pnpm build`
- `pnpm check-types`
- `pnpm docs:generate-meta`
- `pnpm docs:validate-meta`
- `pnpm test`

## Notes

- Visual variant exploration stays in `apps/daisy-showcase`.
- This app focuses on props, usage contracts, and metadata for AI/tooling ingestion.
